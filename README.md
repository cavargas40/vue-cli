# vue-cli

> A Vue.js project

## Build Setup

``` bash
# install vue-cli
npm i -g @vue/cli

# install vue-cli-init
npm i -g @vue/cli-init

# check vue version
vue --version

# init vue project
vue init <template> (e.g. webpack-simple) <project-name> (e.g. vue-cli)

##Inside the project...

# install dependencies
npm i

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
